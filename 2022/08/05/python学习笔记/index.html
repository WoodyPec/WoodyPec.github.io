==第一周==

1.编译和解释

编译是一次性执行全部源代码，后续不再需要源代码，时间久远会有源代码丢失的情况

解释是逐条执行源代码，因此一直需要源代码，更便于维护

2.程序的基本编写方法IPO

Input Process Output

输入是程序的开始

输出是程序展示运算结果的方式，如果没有输出程序是没有意义的

处理方法统称算法，是程序最重要的部分，也是程序的灵魂

3.编程解决问题的步骤（6步骤）

分析问题：分析问题的计算部分

划分边界：划分问题的功能边界，规划IPO

设计算法：设计问题求解的算法，关注问题本身

编写程序：编程

调试测试：调试程序使其正确运行

升级维护：更新完善

精简步骤（3步）

确定IPO：明确计算部分和功能边界

编写程序：将计算求解的设计变成现实

调试程序：确保程序按照正确逻辑能够正确运行

4.为什么要学习编程？

编程能够训练思维

①体现了一种抽象的交互关系、自动化执行的思维模式

②计算思维：区别数学的逻辑思维和物理的实证思维的第三种思维模式

③促进人类思考，增进观察力和深化对交互关系的理解

编程能够增进认识

①不单纯是求解计算问题

②

编程能够带来乐趣

编程能够提高效率

编程可以带来就业机会

5.学习编程的误区

编程很难学吗？掌握方法就很容易！

首先，掌握编程语言的语法，熟悉基本概念和逻辑

其次，结合计算问题思考程序结构，会使用编程套路

最后，参照案例多练习多实践，学会举一反三

6.python的两种编程方式

交互式：通过IDLE（Integrated Development and Learning Environment）对每个输入语句即时运行结果，适合语法练习

文件式：批量执行一组语句并运行结果，是编程的主要方式

6.示例1：温度转换

①缩进

严格明确：缩进是语法的一部分，缩进不正确程序运行错误

所属关系：表达代码间包含和层次关系的唯一手段

长度一致：程序内一致即可，一般用4个空格或1个TAB，缩进长度需要一致

②注释

不被执行，提高程序可读性

单行注释：#

多行注释：‘’’ ‘’’

③命名与保留字

变量：程序中用于保存和表示数据的占位符号，采用标识符来表示

TempStr是变量名字

命名：关联标识符的过程

命名规则：大小写字母、数字、下划线和汉字等字符及组合

**注意事项：**大小写敏感，首字符不能是数字，不能与保留字相同

保留字：被编程语言定义并保留使用的标识符，python共33个保留字

8月7日 9:43更新

④数据类型

程序设计语言不允许存在语法歧义，需要定义数据的形式，如给10,011,101关联一种计算机可以理解的形式

程序设计语言通过一定方式向计算机表达数据的形式，如"123"表示文本字符串123，123则表示数字123

1）字符串：用“”或者‘’都可以

字符串的序号：正向递增序号（从0开始），反向递减序号（最大是-1）

字符串的使用：

​ 索引：返回字符串中单个字符，形式是<字符串>[M]

​ e.g. “请输入带有符号的温度值：”[0]或者TempStr[-1]

​ 切片：返回字符串中的一段字符子串，形式是<字符串>[M:N]

​ e.g. “请输入带有符号的温度值：”[1:3]或者TempStr[-1]，不包含后面的

2）数字类型：整数对应数学中的整数，浮点数对应实数（带有小数部分）

3）列表：使用[]表示，用逗号,分隔各元素

使用保留字in判断一个元素是否再列表中

e.g. TempStr[-1] in [‘C’,‘c’] 判断前者是否与列表中的某个元素相同，相同则返回True

⑤语句和函数：赋值语句，分支语句

赋值语句：给变量赋予新的数据值，右侧的数据类型同时作用于变量

分支语句：如if TempStr[-1] in [‘F’,‘f’]: #如果条件为True则执行冒号后语句

注意：:的作用是如果中间的条件成立，那么则执行冒号后带有缩进的语句，反之跳过缩进的语句

**函数：**根据输入参数产生不同输出的功能过程，采用<函数名>(<参数>)方式使用

e.g. eval(TempStr[0:-1]) #TempStr[0:-1]是参数

eval()函数，也叫评估函数，可以去掉参数最外侧引号并执行余下语句

eval(<字符串或字符串变量>)

e.g. eval(“1”) #输出结果是1 eval(“1+2”) #输出结果是3 eval(‘“1”’) #输出结果是‘1’

eval('print(“Hello”) ') #输出结果是Hello

⑥输入输出

输入函数input()，格式<变量> = input(<提示信息字符串>)

用户输入的信息以字符串类型保存在<变量>中

输出函数print()，格式print(<拟输出字符串或字符串变量>)

此外，print函数的格式化：

e.g. print(“转换后的温度是{:.2f}C”.format©)

{}表示槽，后续变量填充到槽中，{:.2f}表示将变量C填充到这个位置时取小数点后2位

1.利用turtle库绘制蟒蛇

标准库与第三方库，标准库随解释器直接安装，而第三方库需要用户手动安装

库library 包package 模块module

2.turtle库的函数

①窗体布局

setup函数，setup(width,height,startx,starty)，后两个参数是可选的，未指定默认窗口在屏幕中心

②空间坐标体系

绝对坐标：以海龟为中心，参考平面直角坐标系，常用函数goto(x,y)

海龟坐标：以海龟为第一视角，常用函数fd(d)意思是forward，bk(d)意思是backward

circle(r,angle)是以海龟左侧某个点为圆心绘制曲线，想起了曲率。。

③角度坐标体系

绝对角度：以海龟为中心，参考平面直角坐标系，常用函数seth(angle)，注：seth只改变方向不行进

海龟角度：以海龟为第一视角，常用函数left(angle)，right(angle)

④RGB色彩模式

默认采用小数值，可切换为整数值

colormode(mode)，colormode(1.0)表示小数值，colormode(255)表示整数值

⑤画笔控制函数

penup()别名pu，抬起画笔

pendown()别名pd，落下画笔

pensize(width)别名width(width)，画笔宽度

pencolor()画笔颜色，有三种形式

1)颜色字符串，注意小写，如turtle.pencolor(‘purple’)

2)RGB小数值，如turtle.pencolor(0.63,0.13,0.94)

3)RGB元组值，如turtle.pencolor((0.63,0.13,0.94))

⑥运动控制函数

forward(d)别名fd，d表示行进距离，单位是像素

circle(r,extent=None)，默认圆心在海龟左侧距离为r的地方，不指定extent（角度）默认绘制整圆

注：若r为负值则向右侧绘制圆

⑦方向控制函数

setheading(angle)别名seth()，left()，right()

3.库引用import的使用

①<库名>.<函数名>(<函数参数>)

②每次都写库名很麻烦，可以from<库名>import<函数名>或者import*，这样不需要每次都写库名了

比较

方法①不会重名，方法②可能会（与自定义函数）重名

③import<库名>as<库别名>（小名）这个方法最好

4.循环语句

for <变量> in range (<参数>) 参数就是循环的次数，变量表示循环计数（从0开始）

​ <被循环执行的语句>

range(M,N)，从M到N-1，产生N-M个整数

输出结果在t和i之间有空格

==第三周==

本周讲解基本数据类型

1.整数类型

无范围限制，4种进制表示形式

①十进制

②二进制 以0B或0b开头，如0b101

③八进制 以0O或0o开头，如-0o456

④十六进制 以0X或0x开头，如-0x9a

2.浮点数类型

有范围限制，可忽略

浮点数运算存在不确定尾数，不是bug，是由于十进制和二进制不是完全对应引起的，如0.1会对应无限循环的二进制数，于是就会出现这种情况

此时可以使用round(x,d)函数，对x四舍五入，d是小数截取位数

科学计数法：e表示a*10b

3.复数（python独有）

虚数单位用**j**来表示，z.real表示实部，z.imag表示虚部

4.数值运算操作符

基本操作符：+，-，*，/，//，%，**

二元操作符的增强操作符：x op y 其中，op为二元操作符

混合运算会生成“最宽”的数据类型

5.数值运算函数

abs(x)对x取绝对值

divmod(x,y)商余，即同时输出商和余数(x//y,x%y)

pow(x,y[,z])幂余，(x**y)%z，函数中的[]表示可省略，作用：计算更迅速

round(x[,d])默认取整

max(x1,x2,…,xn)

min(x1,x2,…,xn)

数值转换函数

int(x)

float

complex，注：（）里是字符串也可

例子：天天向上的力量

计算思维是抽象和自动化相结合的结果

其中抽象是指抽象一个问题的形式化逻辑，自动化是指将抽象的结果通过程序自动完成

6.字符串类型及操作

2类共4种表示方法，注：多行字符串用’''或"""括起来，可以当作注释

作用：使得字符串中可以出现’和"，聪明的设计

序号：正向递增或反向递减

①索引 格式：<字符串>[M]

②切片 格式：<字符串>[M:N]

高级用法：<字符串>[M:N]，M缺失表示至开头，N缺失表示至结尾

如：“〇一二三四五六七八九十”[:3]结果是"〇一二"

<字符串>[M:N:K]，以K为步长进行切片

如：“〇一二三四五六七八九十”[1:8:2]结果是"一三五七"

字符串逆序，用[::-1]来实现

如：“〇一二三四五六七八九十”[::-1]结果是"十九八七六五四三二一〇”

转义符\反斜杠：表达特定字符的本意

常用的\b回退（一个位置），\r回车（光标移至本行首），\n换行（光标移至下行首）

7.字符串操作符

x + y 连接两个字符串

n * x或x * n 复制n次字符串x

x in s 如果x是s的子串，返回True，否则返回False

8.字符串处理函数

len(x)，str(x)与eval(x)对应，hex(x)形成16进制字符串，oct(x)形成8进制字符串

chr(u)，u为Unicode编码，返回其对应的字符，ord(x)，x为字符，返回其对应的Unicode编码

print函数中end的用法，end=""引号为空，则输出后不换行，默认换行

8.字符串处理方法

”方法“在编程中是一个专有名词，特指.()风格中的函数()是面向对象编程中的特有方法

①一些以方法形式提供的字符串处理功能

str.lower()或str.upper()，返回字符串副本，全部小写/大写

str.split(sep=None)，返回一个列表，由str根据sep被分隔的部分组成
如：“A,B,C”.split(“,”)结果为[‘A’,‘B’,‘C’]，即用,分割字符串形成列表

str.count(sub)，返回子串sub在str中出现的次数

str.replace(old,new)，返回字符串str副本，所有old子串被替换为new
如：“python”.replace(“n”,“n123.io”)结果为"python123.io"

str.center(width[,fillchar])，字符串str根据宽度width居中，fillchar可选
如：“python”.center(20,“=”)结果为=======python=======，左右各7个=

str.strip(chars)，从str中去掉在其左侧和右侧chars中列出的字符
如：“=python=”.strip(“=npo”)结果为"yth"

str.join(iter)，在iter变量除最后元素外每个元素后增加一个str

如：“,”.join(“12345”)结果为"1,2,3,4,5"，主要用于字符串分隔等

9.字符串类型的格式化

使用.format()方法，<模板字符串>.format(<逗号分隔的参数>)

**槽**相当于一个占位信息符，使用{}来表示，只在字符串中有用

槽内部对格式化的配置方式：{<参数序号>:<格式控制标记>}

引导符号是”:“，6种格式控制标记分别是

1)<填充>  2)<对齐>，<左对齐，>右对齐，^居中  3)<宽度>，槽设定的输出宽度

4)<,>，数字的千位分隔符  5)<.精度>，浮点数小数精度或字符串最大输出长度

6)<类型>，整数类型b,c,d,o,x,X,浮点数类型e,E,f,%

填充、对齐、宽度3个一组

eg：“{0:=^20}”.format(“python”) #0可省略

结果：=======python=======

后3个为一组，都和数字有关

eg1：“{:,.2f}”.format(12345.6789)

结果：12,345.68 #只分隔整数部分

eg2：“{0:b},{0:c},{0:d},{0:o},{0:x},{0:X}”.format(425)

结果：110101001,Ʃ,425,651,1a9,1A9

eg3：“{0:e},{0:E},{0:f},{0:%}”.format(3.14)

结果：3.140000e+00,3.140000E+00,3.140000,314.000000%

10.time库的使用

time库包含3类函数，分别是

时间获取time()，获取时间戳。ctime()，获取字符串形式的当前时间。gmtime()，其他程序可利用的格式。

时间格式化strftime(tpl,ts)，strptime(str,tpl)，两个函数是互补的关系

eg1：time.strftime(“%Y-%m-%d %H:%M:%S”,time.gmtime()) 结果：2022-10-05 07:24:20

eg2：

格式化字符串：%Y年；%m月份；%B月份名称；%b月份名称缩写；%d日期；%A星期；%a星期缩写； %H24小时制；%I12小时制；%p上午下午；%M分钟；%S秒。

程序计时sleep()，产生时间，可用于程序休眠。

perf_counter()，测量时间但是起点不确定，连续调用，计算差值才有意义

==第四周==

程序的控制结构分为三种，分别是顺序结构、分支结构和循环结构

1.分支结构

①单分支结构，if “如果，则”

②二分支结构，if else

二分支的紧凑形式，<表达式1> if <条件> else <表达式2>

注：二分支紧凑形式仅支持表达式，不支持带赋值符号的形式

③多分支结构，if elif else

注意：1）多条件之间的包含关系 2）变量取值范围的覆盖

④条件判断及组合

⑤异常处理

基本用法：try…except

高级用法：try<语句块1>except<语句块2>else<语句块3>finally<语句块4>

首先执行try后的语句，若异常执行except；若不异常执行else，最后执行finally

总结：

1)多分支条件之间的覆盖是重要问题

2)程序可运行但不正确，要注意多分支

3)分支结构是程序的重要框架，读程序先看分支

2.循环结构

遍历循环for，无限循环while

①遍历循环

计数循环for in range 字符串循环for c in s 列表遍历循环for item in ls 文件遍历循环for line in fi(fi是文件标识符)

②无限（条件）循环

③循环控制保留字

break跳出并结束循环，执行循环后面的语句

continue结束当此循环，执行后续循环

④循环的高级用法

在循环后加else语句，在循环不被break的情况下执行，若被break则不执行

3.random库

random库是python的标准库

伪随机数：采用梅森旋转算法产生随机序列

基本随机数函数：seed()，random()
扩展随机数函数：randint()，getrandbits()，uniform()，randrange()，choice()，shuffle()
